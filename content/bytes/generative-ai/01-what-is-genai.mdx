# What is Generative AI?

நீங்க ChatGPT, Midjourney, Claude-ன்னு கேள்விப்பட்டிருப்பீங்க. இதெல்லாம் Generative AI tools தான்.

## Traditional AI vs Generative AI

**Traditional AI**: Classifies, predicts, recognizes
- Is this email spam? (Yes/No)
- Will this customer buy? (Probability)
- What's in this image? (Cat, Dog, etc.)

**Generative AI**: Creates new content
- Write a blog post
- Generate an image
- Compose music
- Create code

## How Does It Work?

GenAI models are trained on massive amounts of data (text, images, code) and learn patterns. When you give a prompt, they generate new content based on those patterns.

Think of it like this: நீங்க பத்து ஆயிரம் recipes படிச்சிட்டு, ஒரு புதுசா recipe create பண்றீங்க. Same concept, but with way more data!

## Types of GenAI Models

1. **Text Models** (LLMs - Large Language Models)
   - ChatGPT, Claude, Gemini
   - Generate text, code, translations

2. **Image Models**
   - DALL-E, Midjourney, Stable Diffusion
   - Create images from text descriptions

3. **Audio Models**
   - Generate music, voices, sound effects

4. **Video Models**
   - Sora, Runway
   - Create videos from text

## Real-World Applications

- Content creation (blogs, social media)
- Code generation and debugging
- Customer support chatbots
- Design and creative work
- Data analysis and insights
